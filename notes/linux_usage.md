### 1. 压缩相关

#### 1.1 压缩当前目录下文件夹/文件

```bash
tar -zcvf <filename>.tar.gz <filename>
```

#### 1.2 解压缩当前目录下文件

```bash
tar -zxvf <filename>.tar.gz
```

> 参数解释
>
> 以下参数必选一个
>
> * -c 建立压缩档案
> * -x 解压
> * -t 查看内容
> * -r 向压缩归档文件末尾追加文件
> * -u 更新原压缩包中的文件
>
> 以下参数为可选
>
> * -z 有gzip属性
> * -j 有bz2属性
> * -Z 有compress属性
> * -v 显示所有过程
> * -O 将文件解开到标准输出
>
> -f 必选 最后一个参数，后面接档案名
